## 二、统一风格

### 命名规范

- 文件结构命名

```js
// 文件命名
helloWorld
// 组件命名
HelloWorld.vue
```

- CSS 命名

```js
<!-- HTML -->
jdc.html
jdc_list.html
jdc_detail.html

<!-- SASS -->
jdc.scss
jdc_list.scss
jdc_detail.scss
```

ClassName 的命名应该尽量精短、明确，必须以字母开头命名，且全部字母为小写，单词之间统一使用下划线 “\_” 连接
