# Chrome 扩展

Chrome Extensions，Chrome 浏览器扩展或者叫插件，可通过自定义界面、观察浏览器事件和修改网络来提升浏览体验

使用 HTML、CSS、JS 即可构建。

也可结合 Vite/Webpack 、 Vue/React、Pinia/Zustand、Naive-ui/Ant Design 等技术栈开发 Chrome 插件
